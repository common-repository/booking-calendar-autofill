== Description ==

This plugin works with the Booking Calendar plugin and autofills fields, like "First Name," "Last Name," "Email" and "Phone Number" according to the user who's currently logged in. 

== Installation ==

1. Upload `booking-autofill.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. The plugin will automatically look for the appropriate fields and insert content if it exists
